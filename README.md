# Lifelink-online-organ-donation-mangement-system-
Project Name: Lifelink-online Organ Donation System
Status: Under Development
Description: A system designed to connect organ donors and recipients to improve emergency medical response and coordination.
It is a PHP/MySQL web platform bridging organ donors and recipients via smart matching logic (blood group &amp; urgency). Key features include automated donor reservation, FPDF-generated medical reports, and an admin hospital review simulation. It streamlines healthcare logistics through structured data and real-time status tracking.
